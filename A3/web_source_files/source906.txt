<div class="wikidoc"><b>Project Description</b><br>
Reg2CI is a Command-Line Tool to convert .POL &#40;Policy&#41; or .REG Files &#40;Registry&#41; into System Center Configuration Manager CI&#39;s &#40;Configuration Items&#41;.<br>
<h1>V0.7.0.6 New: Support for .pol (GPO or local Policy Files)</h1>
<h1>Current Features</h1>
<ul>
<li>Convert the .POL or .REG File to a Single Setting CI which is using PowerShell to detect and remediate the Settings
</li><li>Supported Registry DataTypes: DWord, QWord, string </li><li>Option to convert Reg and run CI as X86 process </li></ul>
<h1>Usage</h1>
Command-Line Parameters:<br>
<pre> Reg2CI.exe &lt;Reg File&gt; &lt;Cab File&gt; &lt;Name of the CI&gt; [/X86]
 Reg2CI.exe &lt;POL File&gt; &lt;Cab File&gt; &lt;Name of the CI&gt; [/X86] [/USER] </pre>
<br>
<h1>Examples and Templates:</h1>
<a href="https://reg2ci.codeplex.com/documentation?referringTitle=Home">Documentation</a></div>
