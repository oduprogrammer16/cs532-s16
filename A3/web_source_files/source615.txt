            <p >

                أعلنت قيادة العمليات المشتركة في العراق مقتل أكثر من 80 عنصرا من تنظيم الدولة، وتحرير 68 مدنيا من قبضة التنظيم في الرمادي، غربي العراق.<br />
<br />
وذكر بيان لقيادة العمليات إن قوات قيادة عمليات الأنبار أحبطت هجوماً في منطقة الجرايشي والبو ذياب، أسفر عن مقتل العشرات من تنظيم الدولة وتدمير أسلحتهم ومعداتهم.<br />
<br />
وأشار البيان إلى قيام جهاز مكافحة الإرهاب بتحرير 68 مدنيا، كانوا محتجزين من قبل تنظيم الدولة وإخلائهم بالتنسيق مع شرطة طوارئ المحافظة.<br />
<br />
وفي سياق متصل، أعلنت قيادة شرطة ديالى عن ضبط مخبأ كبير للعبوات الناسفة في عملية أمنية نفذت شمال بعقوبة، شمال شرق العراق.<br />
<br />
وقال العقيد غالب عطية الناطق الإعلامي باسم قيادة شرطة ديالى، اليوم، إن قوة أمنية من الشرطة نفذت عملية أمنية عند الضواحي الشمالية لحوض العظيم شمال بعقوبة، أسفرت عن العثور على مخبأ كبير للعبوات الناسفة ودراجة نارية مفخخة.<br />
<br />
وأضاف أن محتويات المخبأ جرى نقلها إلى مكان آمن، تمهيدا لإبطال مفعولها من قبل الجهات الأمنية المختصة. &nbsp;<br />

<script>
$(function() {
var labels = ['اسبوع', 'يوم', 'ساعة', 'دقيقة', 'ثانية'],
      nextYear =  '2015/01/15 6:00:00 PM',
      template = _.template($('#main-example-template').html()),
      currDate = '00:00:00:00:00',
      nextDate = '00:00:00:00:00',
      parser = /([0-9]{2})/gi,
      $example = $('#main-example');
    // Parse countdown string to an object
    function strfobj(str) {
      var parsed = str.match(parser),
        obj = {};
      labels.forEach(function(label, i) {
        obj[label] = parsed[i]
      });
      return obj;
    }
    // Return the time components that diffs
    function diff(obj1, obj2) {
      var diff = [];
      labels.forEach(function(key) {
        if (obj1[key] !== obj2[key]) {
          diff.push(key);
        }
      });
      return diff;
    }
    // Build the layout
    var initData = strfobj(currDate);
    labels.forEach(function(label, i) {
      $example.append(template({
        curr: initData[label],
        next: initData[label],
        label: label
      }));
    });
